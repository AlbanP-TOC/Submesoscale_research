# Submesoscale research
This repository archives codes related to the study of submesoscale processes. 

## Analytical simulations :
Study focused on the behavior of particles in an analytical vortex.

## Submesoscale simulations !
Experimentation of various forcing and postprocessing

## Vortex detection :
Simple vortex detection algorithm to track vortexes positions in OGCMs

## Mode decomoposition
Experiment of Proper Orthogonal Decomoposition (POD) and Dynamic Mode Decomposition (DMD) on OGCMs data
